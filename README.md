# Forecast - view weather
[![xc compatible](https://xcfile.dev/badge.svg)](https://xcfile.dev)

Cool project

## Tasks

### run

Just run the project

```bash
go run main.go
```

### gen

Generate templ and css

```bash
templ generate
npx tailwindcss -i styles.css -o static/styles.css
```

### dev

```bash
templ generate
npx tailwindcss -i styles.css -o static/styles.css
go run main.go
```
